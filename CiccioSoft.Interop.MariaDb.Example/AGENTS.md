# Istruzioni per il progetto CiccioSoft.Interop.MariaDb.Example

## Scopo

- Questo progetto è un esempio di utilizzo del layer `CiccioSoft.Interop.MariaDb`.
- Serve a mostrare come aprire connessioni native e usare l’API di base.

## Cose importanti

- Mantieni il codice semplice e didattico.
- Non implementare logica di business complessa qui; il progetto è un esempio di uso.

## Comandi utili

- Eseguire l’esempio:
  - `dotnet run --project CiccioSoft.Data.MariaDbEmbedded.Interop.Example/CiccioSoft.Data.MariaDbEmbedded.Interop.Example.csproj`

## Lingua

- Rispondi in italiano quando discuti di questo progetto.
